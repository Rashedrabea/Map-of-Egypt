📖 عن التطبيق
خرائط مصر المتكاملة هو تطبيق خرائط ويب متطور يهدف إلى توفير تجربة خرائط شاملة للمستخدمين في مصر. يدمج التطبيق بين خرائط OpenStreetMap وقوة الذكاء الاصطناعي لتقديم تجربة فريدة ومتكاملة.

🚀 طريقة التشغيل
الطريقة 1: فتح مباشر
قم بتحميل ملف index.html

افتحه في أي متصفح حديث

استمتع بالتطبيق!

الطريقة 2: باستخدام Live Server
قم بتثبيت إضافة Live Server في VS Code

افتح مجلد المشروع

انقر بزر الماوس الأيمن على index.html

اختر "Open with Live Server"

الطريقة 3: النشر على الإنترنت
عبر GitHub Pages:
ارفع الملفات إلى مستودع GitHub

اذهب إلى Settings > Pages

اختر الفرع الرئيسي (main)

سيظهر رابط التطبيق خلال دقيقة

عبر Netlify:
اذهب إلى Netlify.com

اسحب مجلد المشروع إلى الموقع

سيتم النشر تلقائياً

🔒 الخصوصية والأمان
جميع البيانات محفوظة محلياً في متصفحك

لا يتم إرسال أي بيانات إلى خوادم خارجية (عدا خدمات الخرائط والطقس)

لا يوجد تتبع أو جمع للبيانات الشخصية

الموقع الجغرافي لا يتم مشاركته مع أي طرف ثالث
